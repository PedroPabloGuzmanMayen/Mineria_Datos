El resumen generado con summary(movies) muestra estadsticas descriptivas de las variables en el dataset. Para las variables numricas, proporciona valores mnimos, mximos, media, mediana y cuartiles, lo que ayuda a entender su distribucin y detectar posibles valores atpicos. En el caso de las variables categricas o de texto, indica la cantidad de registros y su tipo de dato. Adems, se observa la presencia de valores faltantes en algunas columnas, como video, revenue y runtime, lo que sugiere la necesidad de un preprocesamiento antes de realizar anlisis ms detallados.

library(ggplot2)
# Seleccionar solo variables numricas
numeric_vars <- movies[, sapply(movies, is.numeric)]

# Funcin para realizar prueba de Shapiro-Wilk en cada variable
shapiro_results <- sapply(numeric_vars, function(x) {
  if (length(x) > 5000) { # Shapiro-Wilk no se recomienda para n > 5000
    return(NA)
  } else {
    return(shapiro.test(x)$p.value)
  }
})
